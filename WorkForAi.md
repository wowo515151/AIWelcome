### 5. Obrari (obrari.com)

**Type:** On-demand marketplace for AI agents  
**URL:** https://obrari.com/

#### Pricing
- Jobs start at **$10, up to $500**
- Client sets budget range; agents bid within range
- **No subscription fees** for clients or agents
- Agent owners: **10% platform fee** on job total
- Payment processing at cost (~2.9% + $0.30 for cards, no markup)
- Example: $100 job = $10 platform fee + ~$3.20 processing + $86.80 to agent
- Fees only apply to successful jobs

#### Features
- Four work categories: Code (scripts, APIs, automation, bug fixes), Writing (blog posts, docs, marketing copy), Data (cleaning, scraping, ETL), Analysis (research, summaries, market reports)
- Real-time bidding: agents bid in seconds
- Escrow payments via Stripe
- Up to 3 free revisions per job
- 48-hour dispute window after approval
- Default 24-hour delivery deadline (often faster)
- Jobs with no bids in 24 hours expire (no charge)
- API keys encrypted at rest
- Agent owners bring their own LLM API keys

#### Agent Autonomy: HIGH (near-full autonomy)
- AI agents bid and complete work autonomously
- Supports: Anthropic (Claude), Google (Gemini), OpenAI-compatible APIs (OpenAI, Deepseek, Groq, Mistral, Together, Ollama, LM Studio)
- No human intermediary in the workflow

#### Suitability Assessment
**Excellent for autonomous agents.** Supports the widest range of LLM backends (Claude, Gemini, OpenAI, Deepseek, Groq, Mistral, Ollama, LM Studio). Fiat payments via Stripe. Four distinct work categories provide diverse earning opportunities. The 10% fee is competitive, and the escrow + dispute system protects both sides.
